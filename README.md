Barbería Chilena – Landing Page
Proyecto desarrollado para la asignatura de Programación Back End, correspondiente a la Evaluación N°1: construcción de una Landing Page individual utilizando HTML5 y CSS3 puro, sin frameworks.
Descripción del proyecto
Barbería Chilena es una landing page para una barbería urbana moderna. El sitio presenta la identidad de la marca, los servicios ofrecidos, una lista de precios, la presentación del equipo de barberos, testimonios de clientes y un formulario de contacto para solicitar reservas de hora.
Secciones incluidas
-	Inicio (Hero): presentación principal con llamado a la acción para reservar hora.
-	Sobre nosotros: descripción del negocio y su propuesta de valor.
-	Servicios: cuatro servicios principales con imagen, nombre y descripción.
-	Precios: tabla con los servicios y sus tarifas.
-	Equipo: presentación de los barberos del local.
-	Testimonios: opiniones de clientes.
-	Contacto: formulario con validación para nombre, correo, asunto y mensaje.
-	Footer: datos de contacto y redes sociales del negocio.
Estructura de carpetas
barberia-urbana/
-	index.html – página principal del sitio.
-	src/css/styles.css – hoja de estilos del sitio, incluye variables CSS y media queries.
-	src/js/script.js – validación del formulario de contacto.
-	src/assets/ – imágenes utilizadas en el sitio (logo, servicios y equipo).
.gitignore – archivos y carpetas excluidos del control de versiones.
Tecnologías utilizadas
-	HTML5
-	CSS3 (variables, Flexbox, Grid y diseño responsive)
-	JavaScript (validación del formulario de contacto)
Cómo visualizar el proyecto
1.	Clonar o descargar el repositorio.
2.	Abrir el archivo index.html directamente en el navegador.
3.	Alternativamente, acceder a la versión publicada mediante GitHub Pages.
